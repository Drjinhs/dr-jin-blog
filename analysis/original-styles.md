# Source style analysis

Because the Lovable preview requires workspace authorization, exact computed styles were unavailable. The rebuild uses visual cues from the provided portraits and the brief's expected professional personal-site character.

## Design system

- Palette: deep navy `#102a43`, warm ivory `#f6f2ea`, coral `#ef6f61`, white, and muted blue-grey.
- Typography: a high-contrast editorial serif stack for headings and a neutral system sans-serif for navigation and body copy.
- Layout: maximum content width of 1180px, asymmetric editorial grids, strong whitespace, hairline rules, and large type.
- Shape: predominantly square image frames and cards, with circular accents used for milestones.
- Motion: short entrance transitions and restrained hover lift, disabled under `prefers-reduced-motion`.

All reusable values are recorded in `assets/css/variables.css`.
